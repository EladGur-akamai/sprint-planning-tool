# Feature: Retro Tool

**Priority:** Medium
**Status:** Proposed
**Created:** 2025-7-12

## Description
Add a tool to handle retro for a sprint, and present the insights from the last retro in the 'Sprint Planning' tab.
When filling a retro each team member should provide it's own input, each input should be a lesson learned or todo.
In the Sprint planning tab show all the insights in a nice summerized way.

## Acceptance Criteria
- User is able to fill a retro 
- User is able to select a team and view it's sprints and everything according to his choise

## Technical Considerations
- make it cool, with the ability to chose the team from a drop down or something that shows the logo for each and the chsen one
